<h2 id="how-it-works">ğŸ“ How It Works</h2>

Atelier do Sabor's website is a modern, responsive platform that showcases the restaurant's offerings:

1. **Dynamic Content**: The website loads restaurant services and menu items using AJAX from an external API.
2. **Responsive Design**: The layout adapts to various screen sizes, ensuring a seamless experience on desktop and mobile devices.
3. **Interactive Elements**: Users can easily navigate sections like the menu, services, events, and contact form.
4. **Contact Form**: A functional contact form allows users to reach out via email or phone.
